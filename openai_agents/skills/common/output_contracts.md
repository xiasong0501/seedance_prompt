# Output Contracts

## Director Analysis Contract

Must include:

- story structure summary
- beat-by-beat director breakdown
- character table
- location table
- material state: 新增 / 复用 / 变体

## Art Contract

Character prompts must:

- support reference image generation
- clearly define face, body, hair, clothing, accessories, shoes, temperament
- state the layout for design-sheet style output

Scene prompts must:

- support multi-panel grid generation
- define space, lighting, palette, texture, props, mood
- keep visual consistency across all panels

## Storyboard Contract

Must:

- map each beat to one prompt block
- use director intent as the primary source of truth
- integrate assets via references where appropriate
- remain executable for Seedance-style video generation

## Explosive Rewrite Contract

Must:

- score current variants
- explain strengths and weaknesses
- explain the rewrite plan
- explicitly list where explosive upgrades were inserted
- preserve core story continuity
