# Storyboard Workflow Skill

## Purpose

Produce the strongest possible `02-seedance-prompts.md`.

## Inputs

- `01-director-analysis.md`
- `character-prompts.md`
- `scene-prompts.md`
- methodology and generation constraints

## Prompt Construction Rules

- one beat should map to one clear generation task
- use dynamic, executable visual language
- preserve the director’s emotional and visual intent
- do not overload a single prompt with too many unrelated actions
- use asset references consistently when available

## Quality Standard

- movement should be physically plausible
- timing density should be reasonable
- transitions should be clear
- prompts should remain generation-friendly rather than literary-only
