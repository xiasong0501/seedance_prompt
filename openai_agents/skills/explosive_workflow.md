# Explosive Workflow Skill

## Purpose

Compare multiple same-episode script variants and produce the strongest version for retention, emotional pull, and next-episode click-through.

## Evaluation Dimensions

- opening hook
- conflict arrival speed
- emotional pressure
- pace efficiency
- protagonist desire clarity
- end-of-episode cliffhanger force

## Required Outputs

For each episode:

- current scoring report
- strengths
- weaknesses
- rewrite plan
- explicit “where explosive upgrades were inserted”
- rewritten script

## Rewrite Principles

- preserve core plot logic
- sharpen rather than randomize
- frontload the hook
- reduce explanatory drag
- increase desire, danger, shame, contrast, or urgency where appropriate
- strengthen the last beat before cut
- avoid over-sexualized or unsafe escalation
