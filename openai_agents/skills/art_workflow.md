# Art Workflow Skill

## Purpose

Produce the strongest possible:

- `assets/<剧名>-gpt/character-prompts.md`
- `assets/<剧名>-gpt/scene-prompts.md`

## Primary Input Priority

1. director analysis
2. existing same-series assets
3. episode analysis / series context

## Character Prompt Standard

Every character prompt must cover:

- age and identity feeling
- face structure and features
- hair
- body type
- costume silhouette
- fabric and material
- accessories
- shoes
- overall temperament
- design-sheet layout requirements

## Scene Prompt Standard

Every scene grid must cover:

- layout and space
- time of day
- lighting direction and color
- palette
- material texture
- props and anchor objects
- depth and layering
- mood

## Production Rule

- only write new or variant entries when possible
- do not duplicate reusable history
