# Director Workflow Skill

## Purpose

Convert a script into the strongest possible `01-director-analysis.md`.

## Required Process

1. Read the script carefully
2. Identify natural story beats
3. Assign a practical duration suggestion for each beat
4. Write director-oriented scene narration for each beat
5. Extract production-ready character and location tables
6. Mark each character/location as 新增 / 复用 / 变体

## Beat Quality Requirements

Each beat should include:

- who is present
- where they are
- what physically happens
- what emotional shift occurs
- camera feel
- light/color/mood

## Director Analysis Standard

- beats should be shootable / generatable
- beat boundaries should follow narrative purpose, not arbitrary line breaks
- the document should directly help art and storyboard stages
