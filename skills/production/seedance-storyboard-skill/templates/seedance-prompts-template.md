# Seedance 2.0 提示词产出模板

输出文件：outputs/<剧名>/<集数>/02-seedance-prompts.md

---

## 输出格式

```markdown
## 素材对应表

| 引用编号 | 素材类型 | 对应素材 |
|---------|---------|---------|
| @图片1 | 人物参考 | [人物名，参见 assets/<剧名>/character-prompts.md] |
| @图片2 | 人物参考 | [人物名，参见 assets/<剧名>/character-prompts.md] |
| @图片3 | 场景参考 | [场景名，参见 assets/<剧名>/scene-prompts.md] |

---

## P01 [剧情标题]

- 建议时长：[8秒 / 10秒 / 15秒]
- 主要引用：[@图片1 @图片3]
- 承接关系：[上一条分镜怎么进入本条、结尾如何交给下一条]
- 声音设计：[背景氛围、人声、静默设计]

**统一复合提示词（主时间线）**：
- [0.0-2.0秒｜画面：建立空间、主体在镜与首个动作触发]
- [2.0-4.0秒｜画面：动作推进或镜头转向｜对白窗：2.6-3.5秒 角色A（语气）“...”]
- [4.0-6.0秒｜画面：反应、结果或尾帧钩子｜声音：脚步/衣料/风压/器物轻震]

可直接投喂正文：
[由同一条主时间线自动整合出的 Seedance 2.0 动态提示词。
不再单独创造第二套“镜头节拍 / 对白时间线 / prompt 正文”时间系统。
正文必须和上方主时间线完全同源，可直接复制到 Seedance 2.0 生成视频。]

---
```

---

## @引用语法

使用 Seedance 2.0 原生 @引用语法：
- @图片1、@图片2... → 参考图片（人物/场景参考）
- @视频1、@视频2... → 参考视频
- @音频1、@音频2... → 参考音频
- 编号与文档头部素材对应表一一对应

---

## 输出规范

- 中文提示词
- 提示词采用叙事描述式，不要用关键词堆叠式
- 直接输出完整提示词，不要逐条解释设计理由
- 每条提示词必须完整可用，用户可以直接复制到 Seedance 2.0 生成视频
- 台词必须串行安排，不能让两句完整台词同时发生
- `master_timeline` 是唯一主时间线；`shot_beat_plan`、`dialogue_timeline`、`prompt_text` 只能由它派生，不能各写各的
