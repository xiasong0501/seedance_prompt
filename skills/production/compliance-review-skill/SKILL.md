# 内容合规审核技能

[技能说明]
    内容合规审核技能，检查各阶段产出是否触碰 Seedance 2.0 和 Gemini / Nano Banana 的平台内容限制。此 skill 在每次导演审核的第二步使用，与阶段专属的业务审核 skill 配合。

[审核对象]
    根据当前阶段不同：
    - 阶段一：01-director-analysis.md（导演讲戏本、人物清单、场景清单）
    - 阶段二：assets/<剧名>/character-prompts.md 和 assets/<剧名>/scene-prompts.md 中本集新增内容
    - 阶段三：02-seedance-prompts.md（分镜师的 Seedance 2.0 提示词）

[审核流程]

    第一步：读取待审核内容
        - 读取当前阶段的产出文件

    第二步：逐项检查
        按照下方验收清单逐项检查

    第四步：输出结果
        - 全部通过 → PASS
        - 任一未通过 → FAIL + 问题清单

[验收清单]

    真人/名人限制（Seedance + Gemini 共同红线）
        □ 角色参考图是否为 AI 生成的虚构人物（非真人照片）？
        □ 角色外观是否与真实名人/公众人物不相似？
        □ 提示词中是否不包含公众人物姓名？
        □ 提示词中是否不引用真人导演名作为风格参考（如"XX 风格"）？

    版权 IP 限制
        □ 是否不包含已知版权 IP 角色名称（迪士尼、漫威、DC、哈利波特等）？
        □ 是否不包含已知版权 IP 的形象/设定描述？
        □ 是否不试图复刻经典影视片段？

    政治/国家安全（Seedance 平台红线）
        □ 是否不涉及反对宪法基本原则？
        □ 是否不涉及危害国家安全？
        □ 是否不涉及煽动民族仇恨/破坏民族团结？
        □ 是否不涉及歪曲/否定英雄烈士？
        □ 是否不涉及宣扬恐怖主义/极端主义？

    宗教类
        □ 是否不涉及违反宗教政策？
        □ 是否不涉及宣扬邪教/封建迷信？
        □ 是否不生成宗教人物形象（Gemini 限制）？

    色情/低俗
        □ 是否不包含淫秽色情内容？
        □ 是否不包含性暗示/软色情？
        □ 服装设计是否避免过度暴露（降低 Gemini 误杀）？
        □ 动漫/漫画风格是否注意 Gemini 更高的拦截率？

    暴力/血腥
        □ 武打/格斗场景是否使用虚构角色？
        □ 是否不展示/美化过度血腥暴力？
        □ 是否不包含赌博/诈骗/犯罪指导？

    OutputVideoSensitiveContentDetected 高风险规避（阶段三重点）
        □ 是否避免使用直观伤口、喷溅血液、血肉模糊、断肢断头、尸体特写、器官外露、刑讯细节等高风险镜头描述？
        □ 是否避免把“羞辱/压迫/反击”写成可视化的虐待细节、身体损毁细节或高度写实的濒死画面？
        □ 是否避免写“强吻、撕衣、按倒、裸露、湿身贴体、胸腿腰臀特写、喘息呻吟”等软色情或强性暗示画面？
        □ 是否避免写“啐口水、吐唾沫、唾液落地、朝人脸上啐、用身体羞辱对方”等低俗且易触发误杀的侮辱动作？
        □ 是否避免把“纳妾、受用、小妾、春宵、一枝梨花压海棠、捂下腹、染病/传病/恶疾细节、细菌战”等性羞辱或疾病传播联想写成直给对白与镜头重点？
        □ 是否避免在非正式战斗段里提前铺陈 AK、坦克、迫击炮、炮弹落体、火力覆盖、清场、一个不留 等现代战争直白词，导致模型提前生成高危军武暴力画面？
        □ 是否避免把未成年人或年龄模糊角色置于暧昧、受虐、裸露或成人化镜头语境中？
        □ 是否优先用“震退、压制、失去反抗能力、狼狈倒地、情绪失控、场面压迫感”这类非直观结果描述替代具体血腥/色情细节？
        □ 如果剧情必须表现死亡、惩罚、重创、凌辱、献祭等重口风险情节，是否改写为远景、背影、切后结果、环境反应、众人反应、器物碎裂、能量余波等更安全的表达？
        □ 如果剧情必须表现大规模火力、械斗、追杀或清场，是否优先改写为“远景机械齐鸣、强光爆闪、烟尘覆盖、地面震动、众人被震退/扑倒、赛场失控后的结果画面”，而不是逐个击中、逐个爆炸、逐个处决？

    未成年人保护
        □ 是否不涉及任何形式对未成年人的侵害内容？
        □ 是否不涉及危害未成年人身心健康的内容？

    仇恨歧视
        □ 是否不包含基于性别/年龄/地域/民族/种族/宗教的歧视？

    人身权利
        □ 是否不侵犯他人名誉权/隐私权/肖像权/知识产权？

    虚假信息
        □ 是否不涉及散布谣言/歪曲事实？

    Gemini 特有高频误杀规避（阶段二重点关注）
        □ 人物设计是否避免内衣/泳装类描述？
        □ 运动服装描述是否不过于紧身暴露？
        □ 医学/解剖类场景是否有风险提示？

[输出规范]
    PASS 格式：
        ✅ **合规审核：PASS**
        内容未触碰平台红线。

    FAIL 格式：
        ❌ **合规审核：FAIL**

        **问题1**：[具体违规描述]
        - 位置：[如 人物设计 > 角色名 / P03 提示词]
        - 触碰红线：[哪条规则]
        - 相关平台：[Seedance / Gemini / 两者]
        - 修改方向：[如何修改以规避]

        **问题2**：...

        ---
        **风险等级**：
        1. 🔴 必须修改（触碰绝对禁止红线）：[列出]
        2. 🟡 建议修改（可能触发误杀）：[列出]
