# 题材补充 Skill：复仇逆袭

- 先算清屈辱账，再安排逐层回收与公开反噬。
- 这个题材的爽感来自证据、身份、舆论同时翻盘。
- 主角的强要建立在忍耐、判断与精准出手，不是一直怒吼。
- 卡点优先落在底牌露出一角、下一轮更大清算将至的时候。

## 人工审核并入经验

### 结构与钩子
- 首集尽快建立屈辱来源、现实代价和主角无法立刻反击的原因。
- 反打前必须先埋可回收的证据、能力或旧身份，不可硬掰巧合。
- 每轮反制只解决一层压迫，保留更大的清算目标。

### 人物塑造
- 主角的强来自克制、记账和精准出手，不能只靠嘴炮。
- 反派每次出招都应暴露新的弱点、贪念或逻辑漏洞。
- 旧情、亲情、师承、利益链都可以成为复仇路径，而不是无用背景。

### 服化道统一
- 主角前后状态可通过衣着整洁度、材质与配件变化表现翻身节奏。
- 反派的服装和饰物要快速说明其权力来源与阶层位置。
- 清算场避免过多反光和花纹，优先保证表情与证据可读。

### 场景与调度
- 公开空间比私密空间更适合承接羞辱与反打。
- 空间必须留出围观层、主角层和施压层，便于权力翻转。
- 核心证据最好有固定展示点，方便多轮推进与回收。

### 对白与节奏
- 反击台词宜短、准、狠，优先点破身份、事实与后果。
- 真正有力的狠话应放在证据出现之后，而不是提前空放。
- 清算句后要留围观反应，不要急着切走。

### 分镜表达
- 施压时先给高位者占据空间，再让主角通过动作或证据夺回中心。
- 反派受挫后必须有清晰近景反应，爽点才成立。
- 逆袭场优先采用“施压—亮证—反咬—围观反应—拍板”结构。

### 生产红线
- 前期埋下的羞辱点、证据点和代价点必须被后续逐条回收。
- 主角一旦完成某层翻盘，后续不能无故回到完全被压制状态。
- 不要把复仇写成连续巧合或无成本开挂。
