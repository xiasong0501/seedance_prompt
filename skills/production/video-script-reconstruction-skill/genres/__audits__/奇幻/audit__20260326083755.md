# 题材包体检报告：奇幻

- 健康分：42
- 综合结论：该题材包存在明显的包内污染与职责失衡：大量条目带有具体剧集残留（人名、道具、病因、拜帖数量等），并混入现代情感、婚内背叛、医院会诊、权贵打脸等子模板经验，导致“奇幻”题材共性不足、具体场景过强。playbook 与 skill 间存在大量逐字重复，且不少规则更像导演执行细则或单集 continuity 备注，不适合作为正式题材包长期沉淀。优点是已积累一些可复用的奇幻表达经验，如低特效成立方式、时间触发、异常后果卡点、超常能力在日常空间中的压制感。但当前需要先做去具体剧残留、去跨题材模板化、去重复，再补足真正属于奇幻题材的世界规则与能力代价类规则。

## 预检摘要

- playbook 条目数：113
- skill 规则数：122
- 包内疑似重复：12
- 跨题材高重合：4
- 可疑条目：1

## 亮点

- 已沉淀出少量较像奇幻共性的有效经验：‘不依赖重特效’、‘时间触发型反转’、‘异常后果卡点’。
- 对短剧节奏、卡点、镜头信息密度有较强执行经验，可作为 skill 方法论基础。
- 部分条目能把奇幻落到低成本可拍层面，如日常空间中的突然压制、人物状态反差、关键道具前后命名。

## 发现的问题

### F01 [high] cross_genre_overlap

- 位置：playbook.json / skill.md 多处
- 问题：大量条目明显不是‘奇幻’共性，而是现代情感/婚内背叛/医院悬疑/权贵打脸等其他题材或子模板经验，例如‘婚内背叛清算戏优先放在公开家庭空间’、‘病床居中、家属站侧、医生围床’、‘会诊戏优先使用中近景和正反打’、‘女主官宣关系时必须给身体靠近和眼神确认镜头’。
- 影响：这会让题材包失去题材层抽象能力，模型容易把‘奇幻’误写成现代狗血、医院群戏或背叛打脸剧，降低泛化能力。
- 建议：保留真正与奇幻机制相关的规则；将明显依赖‘婚恋背叛/医院/权贵围猎’前提的条目删除，或改写为更高层的奇幻适配规则。
- 置信度：0.99

### F02 [high] missing

- 位置：playbook.json::continuity_guardrails
- 问题：存在大量具体剧集残留：‘叶枫’‘江天雪’‘黄雨晗’‘申家’‘盲杖、墨镜、挂钟、雪糕’‘蛊虫’‘八+一拜帖’等。
- 影响：continuity_guardrails 应是题材级连续性约束，不应绑定某部具体剧。此类内容会严重污染后续生成，使模型误以为这些设定属于奇幻题材默认元素。
- 建议：整批移除具体人名、具体道具、具体病因、具体数量锚点；若其中有可抽象经验，改写为通用规则，如‘已公开的身份/关系/能力变化不可无痕回退’。
- 置信度：1

### F03 [high] duplicate

- 位置：playbook.json 与 skill.md 跨文件多处
- 问题：两文件存在大量逐字重复，预检已列出多条 1.0 重复，如‘命运变量型角色’‘决定已做出、方法未展示’‘拜帖-邀约-赴约’‘关键道具先私密后公共命名’‘集尾卡点优先停在后果将至、解释未给’等。
- 影响：playbook 应偏结构化规则，skill 应偏展开说明。逐字重复会造成维护成本高、冲突难发现，也削弱文件分工。
- 建议：同一句只保留一处：抽象、短、可检索的版本留在 playbook；带解释、适用条件、例外说明的版本留在 skill。
- 置信度：0.98

### F04 [high] structure

- 位置：playbook.json 多字段
- 问题：字段职责混乱。script_hooks、storyboard_focus、director_focus、dialogue_timing_rules、scene_focus 之间多次交叉；例如卡点规则同时出现在 script_hooks、storyboard_focus、director_focus、skill 分镜表达。
- 影响：同类规则散落多处，人工难判断优先级，也容易出现近重复和冲突。
- 建议：将‘卡点/开场30秒/动作三拍/时间触发’统一收束到 storyboard_focus；将镜头执行如‘先拍脸后拍动作’保留在 director_focus；将场景模板与空间锚点留在 scene_focus；script_hooks 只保留题材级开局钩子。
- 置信度：0.95

### F10 [high] missing

- 位置：全包层面
- 问题：真正属于奇幻题材的关键能力缺失：世界规则建立、超常能力边界、代价/限制、常人与异常的认知差、奇幻元素如何改变人物选择，而不只是制造打脸。
- 影响：当前包更像‘现代爽剧里加一点超常’，而不是完整奇幻题材方法论。缺少这些规则会导致生成内容只有异能表演，没有世界感与因果闭环。
- 建议：新增少量核心规则：1) 超常能力首次出现必须同时给出可观察后果；2) 能力使用需有代价/限制； 3) 世界规则揭示遵循‘异常现象—误判—局部验证—新规则成立’；4) 奇幻能力必须改变人物关系或选择。
- 置信度：0.95

### F05 [medium] duplicate

- 位置：playbook.json::character_focus / costume_focus / scene_focus / skill.md 多处
- 问题：存在大量近重复条目，如‘背叛型伴侣…吸引力与危险感’出现两版；‘第三者角色…’出现两版；‘强势家属/掌权型家属’高度相近；‘妆发要服务近景表情读取’在 costume_focus 与 skill 中反复出现；‘三人同场阵营识别’在人物、场景、分镜多处重复。
- 影响：近重复会稀释真正有效的规则，且这些条目大多依赖特定现代情感/医院/三角关系模板，不适合作为奇幻题材核心。
- 建议：每组只保留信息量最高、适用范围最广的一条；删除依赖‘背叛伴侣/第三者/家属/医生’等特定模板的版本。
- 置信度：0.94

### F06 [medium] clarity

- 位置：skill.md::人物塑造
- 问题：存在明显截断文本：‘集尾卡点优先卡在‘关系已发生、后果未揭晓’的瞬间，比单纯人…’。
- 影响：截断文本保真度低，模型无法稳定理解，且该条本身也更像分镜/卡点规则，不属于人物塑造。
- 建议：若能追溯原文则改写完整并移至‘分镜表达’；若无法确认原意则直接删除。
- 置信度：1

### F07 [medium] reasonableness

- 位置：playbook.json::scene_focus
- 问题：‘跪地场景人物间距精确：求援者距主角1.2米，质疑者距主角1.8米’过度量化，像单次拍摄经验，不适合作为题材规则。
- 影响：题材包应提供可迁移原则，不应给出伪精确数值；不同景别、镜头焦段、场地大小下该数值没有稳定适用性。
- 建议：改为原则性表述，如‘求援者与主角保持明显近距低位，质疑者保持更远的侧向高位，形成压迫梯度’。
- 置信度：0.97

### F08 [medium] scope

- 位置：playbook.json::director_focus / dialogue_timing_rules / skill.md 多处
- 问题：若干条目过于通用，几乎适用于所有短剧，不足以证明‘奇幻’特征，如‘冲突升级时要让每句关键台词都推动局势’、‘传播型金句段应减少切镜频率’、‘专业冲突戏优先用群像全景交代站位’。
- 影响：正式题材包容量有限，过多通用短剧规则会挤占真正的奇幻规则空间。
- 建议：保留少量与奇幻表达强相关的通用规则，其余迁移到更通用的短剧基础包，或删除。
- 置信度：0.9

### F09 [medium] missing

- 位置：playbook.json::core_audience_promises / character_design_focus
- 问题：两个关键字段为空，且当前包缺少对‘奇幻观众期待’与‘奇幻角色设计共性’的明确抽象。
- 影响：没有 audience promise，模型难抓住题材爽点；没有角色设计共性，现有内容就会被具体剧模板替代。
- 建议：补充少量高质量规则，聚焦‘异常进入日常、规则逐步揭示、能力有代价、世界反馈真实’等奇幻核心期待。
- 置信度：0.96

### F11 [medium] cross_genre_overlap

- 位置：playbook.json::character_focus / costume_focus / skill.md::人物塑造/服化道统一
- 问题：‘质疑型配角可复用现代混搭西装+油头模板，适用于多集不同家族’、‘墨绿裙装’、‘现代元素与古风环境冲突’等条目带有强烈具体造型模板痕迹，且互相冲突。
- 影响：这类条目既不稳定适用于奇幻，也会把模型引向固定审美模板，甚至造成时代背景混乱。
- 建议：删除具体服装模板与颜色示例，只保留高层原则，如‘质疑者需高辨识度且与主角阵营形成视觉对立’。
- 置信度：0.92

### F12 [medium] scope

- 位置：playbook.json::script_hooks / skill.md::结构与钩子
- 问题：‘奇幻主线推进采用拜帖-邀约-赴约三段式’、‘命运变量型角色’等条目有一定启发性，但更像某类宅门/权贵/命书式子模板，不足以覆盖奇幻整体。
- 影响：若作为题材级主规则，会把奇幻误收窄为少数叙事模板。
- 建议：保留时需明确适用条件，如‘适用于权力邀请/命运召唤型奇幻支线’；否则降级到 skill 中的可选模板，不宜作为 playbook 核心钩子。
- 置信度：0.88
