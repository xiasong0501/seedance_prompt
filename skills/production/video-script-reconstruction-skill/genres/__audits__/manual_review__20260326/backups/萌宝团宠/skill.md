# 题材补充 Skill：萌宝团宠

- 分析时要区分“孩子卖萌”与“孩子推动主线”的桥段。
- 剧本重建要强化认亲、护短、误会破解、关系催化。
- 孩子的对白和动作要承担推动成人关系变化的功能。
- 场景与节奏应兼顾轻喜温情和真相推进。

## 人工审核并入经验

### 结构与钩子
- 团宠感不一定来自所有人宠孩子，也可以来自最强者因孩子而改变立场并出手护短。
- 萌宝权谋题材中，可先让成人角色在公开场合‘看似过关’，再用孩子掌握的真相进行二次否定，制造更强反差。
- 萌宝题材开场可先让孩子说出一句超出年龄的有效判断，迅速建立‘可爱外形+高维认知’反差。
- 温情支付后立刻接新的关系阻力，是萌宝宅门题材高效的集尾卡点方式。
- 萌宝认亲戏可采用‘门槛羞辱—主角克制退让—强者护短—公开命名身份—反派放狠话’五步结构。
- 角色塑造skill：用幼态外形、老成语气、精准毒舌和超常能力四件套快速建立萌宝主角辨识度。
- 认亲题材中，先用旁人转述‘家人一直在找你’制造暖意，再立刻安排现实阻碍，能显著提升追更欲。
- 认亲戏不要只写温情，先给门槛羞辱，再给反打与拥抱，情绪收益更高。
- 短剧门口戏可用‘不让进门’作为具象冲突，再通过破门/越门完成身份反转。
### 人物塑造
- 萌宝的魅力来自‘脸软心硬、嘴毒但不恶毒、会看戏也会抱大腿’。
- 萌宝的魅力来自‘脸越无辜，话越毒辣，且每句都有效’。
- 强者型萌宝更容易形成团宠，因为她先保护别人，别人才会更爱她。
- 萌宝的吸引力来自‘看起来软，处理事却很硬’。
- 陆正南的魅力来自‘不废话、直接处理’的父权护短
### 对白与节奏
- 萌宝心声适合用现代感吐槽词切入，再迅速落到有效信息。
- 萌宝心声要承担最终价值判断，最好一句封喉。
- 萌宝题材里，孩子必须直接处理关键冲突，而不是只在旁边说可爱台词。
- 角色塑造skill：让萌宝在一句台词里同时体现可爱口吻与明确目标，如‘为了我的小命……’，能快速建立高辨识度。
- 对白skill：让萌宝说出带现代感或成人感判断的话，但每句都必须推动信息或价值判断。
- 萌宝台词适合短、准、带一点自我吐槽感，能同时承担人物魅力和信息推进。
- 萌宝吐槽句式要短、准、带轻微嫌弃感，如‘你们大人都这么不讲理吗’。
### 场景与调度
- 萌宝宅门开局可采用‘高规格迎回—短暂情感回暖—门槛羞辱打脸’三段式，快速建立观众站位。
- 开场用市井公共空间最适合快速建立‘被众人认可’的团宠氛围。
- 门口/门槛是认亲题材的高效冲突场，因为它天然承载‘能否进入家庭系统’的象征意义。
- 围观半圆站位非常适合做萌宝中心化构图
- 门槛空间天然适合认亲、拦门、护短、放狠话四类戏。
- 高门大宅门口是极佳的身份冲突场：既能承载礼法，也能承载羞辱与破门反转。
- 祠堂/门前/院内树下这类半公开空间特别适合宅门认亲与验真戏
