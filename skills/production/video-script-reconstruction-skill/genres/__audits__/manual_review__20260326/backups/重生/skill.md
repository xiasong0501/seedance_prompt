# 题材补充 Skill：重生

- 优先识别前世代价、今生目标和提前布局的证据。
- 剧本重建要强化“这一世不再重蹈覆辙”的主动性。
- 重点抓取提前识破、逆向利用、改命节点。
- 情绪设计要兼顾隐痛感和操盘感。

## 人工审核并入经验

### 人物塑造
- 重生宅门戏可先用同辈羞辱快速建立女主低位，再让女主用一句短宣言切换为主动者。
- 女主若并非完全无过错，而是曾做错关键一步，重生改命会更有戏剧张力。

### 结构与钩子
- 重生首集可采用‘前世死亡终局→总账独白→惊醒回到关键日’三段式。
- 重生题材第二集可直接兑现上集卡点，不拖延关键改命动作
- 重生信息不必一次讲全，只需先锁定一个因果锚点即可驱动追看

### 场景与调度
- 重生确认不必复杂特效，空间光色与服装状态反差就足够。
