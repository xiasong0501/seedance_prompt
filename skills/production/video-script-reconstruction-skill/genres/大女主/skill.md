# 题材补充 Skill：大女主

- 大女主的核心不是“身份高”，而是“做决定、扛后果、改局势”。
- 能力展示必须能改变剧情走向，而不是停留在口号。
- 感情线可以存在，但只能为女主成长与选择服务。
- 卡点优先停在女主做出高代价决定、公开改写格局的瞬间。

## 人工审核并入经验

### 结构与钩子
- 首个关键场必须展示女主在压力下做出有后果的主动选择。
- 女主的能力要尽早落在具体问题上，体现她能解决什么、承担什么。
- 每轮成长都应伴随新的代价或更难的局，不可一路无损平推。
- 奇幻宅门题材中，主角的超常能力最好直接解释她为何必须留在这个家，而不是只用来打架
- applicable_scenes：能力暗示
### 人物塑造
- 女主的强不等于永远正确，而是能在错误后继续校准并承担。
- 她的魅力来自判断力、节奏感和边界感，不来自一路被夸。
- 同盟与对手都应逼出她不同层面的主体性。

### 服化道统一
- 造型要服务“行动中的权威”，避免只剩贵气而没有执行力。
- 女主不同时期的材质、线条和色温应体现权力变化与心理阶段。
- 配饰最好承担身份、记忆或决策锚点，而非纯装饰。

### 场景与调度
- 能让女主直接改写结果的空间优先级最高，如主厅、会议室、法场、签约点。
- 空间调度要允许她从被包围、被质疑转为掌握全场。
- 高压空间里必须有清晰主位和观看位，方便表现权力改组。

### 对白与节奏
- 关键表态宜短、准、能承接反应，避免长篇口号化宣讲。
- 反击句最好先落事实，再落选择，最后落代价。
- 真正的成长台词应出现在做出代价后，而不是代价之前。

### 分镜表达
- 女主从边缘位走向中心位的过程必须可视化。
- 高光场优先给思考、决断、执行三段，而不是只拍结果。
- 成长快感来自“忍受—判断—出手—定局”而非空喊宣言。
- 能力揭示戏必须跳轴正反打，打破常规对话轴线以标志话语权转移
### 生产红线
- 女主一旦建立能力标签，后续失败必须给出更强阻力而非强行降智。
- 前期明确的价值选择要影响后续关系与行动，不能空喊后遗忘。
- 不要把大女主写成只会发火或说狠话的空壳强者。
- ‘灵力弱了许多’为能力限制设定，后续恢复必须有明确代价（如献祭记忆）
- 避免让女主在能力揭示前有主动微笑/抬头动作（破坏人设可信度）
