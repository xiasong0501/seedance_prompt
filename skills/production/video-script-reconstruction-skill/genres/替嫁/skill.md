# 题材补充 Skill：替嫁

- 替嫁的第一驱动力是身份错位与暴露风险，不是单纯婚恋。
- 观众要持续担心“她什么时候会被认出来”，同时期待她反向改局。
- 婚礼、洞房、见长辈、交接信物，都是高压试探场。
- 卡点优先落在露馅线索被触碰、婚姻利益被重新分配的时候。

## 人工审核并入经验

### 结构与钩子
- 开局必须说清楚为何替嫁、谁在推动、暴露后果是什么。
- 婚礼、敬茶、洞房、认亲等仪式节点要成为第一轮试探场。
- 主角要尽快遇到一次“差点露馅”的测试，形成持续悬念。

### 人物塑造
- 替嫁主角不是万能表演者，她的破绽与补救同样重要。
- 夫君或核心对手的信任变化应建立在共同经历与证据上。
- 所有重要配角都应围绕“看穿/利用/保护”主角形成不同态度。

### 服化道统一
- 婚嫁服饰、首饰、嫁妆和私人物件必须承担当事人身份锚点。
- 主角造型要能呈现“借来的身份”与“真实的紧绷感”。
- 重要近景优先保证妆发和饰物逻辑连贯，避免一眼穿帮。

### 场景与调度
- 喜堂、洞房、敬茶厅、家门口、账房是替嫁高频有效空间。
- 空间中要保留可核验身份的制度物，如礼单、族谱、信物、床帐。
- 替嫁戏适合在高压半公开空间推进，既能遮掩又容易被撞破。

### 对白与节奏
- 主角的应答要短、稳、留余地，避免说得太满。
- 试探台词要围绕称呼、旧事、规矩和习惯展开，增强露馅风险。
- 真正关键的认同或怀疑句，应放在仪式完成前后形成压力差。

### 分镜表达
- 身份试探场应优先拍主角反应、线索物和观察者眼神。
- 礼仪流程必须拍清楚关键步骤，否则替嫁紧张感不成立。
- 替嫁节奏要在“稳住场面—出现线索—强行圆回—留下后患”之间循环。

### 生产红线
- 替嫁秘密一旦成立，后续每场都要考虑它是否被触碰。
- 已经发生的误认、怀疑、默许或同盟，后续不能无痕清零。
- 不要让全府上下长期失明，替嫁必须有真实识别压力。
