# 题材补充 Skill：穿越

- 穿越的核心抓点是“我不属于这里”，以及如何活下去、藏下去、用起来。
- 信息差与认知错位，要比单纯时代奇观更重要。
- 主角不必立刻懂世界，但必须尽快意识到代价与规则。
- 卡点优先落在身份差点暴露、现代认知首次奏效、原世界线被改写的时候。

## 人工审核并入经验

### 结构与钩子
- 开局必须让观众看到主角不属于当前世界的明确证据。
- 尽快交代宿主身份、时代规则和主角当前最危险的暴露点。
- 第一轮胜利最好来自现代认知、信息差或思维方式，而不是立即变强。

### 人物塑造
- 主角不能瞬间全知全能，学习成本是戏的一部分。
- 他/她的现代认知应改变解决问题的方式，而非只是说新名词。
- 宿主残留关系必须持续给主角制造选择压力。

### 服化道统一
- 造型要体现“这具身体原本属于谁”，不能完全服务主角旧身份。
- 主角的微小穿法习惯可以成为穿越后的异样痕迹。
- 时代、阶层、身份对应的服装逻辑必须清楚，才能承接错位。

### 场景与调度
- 寝室、梳妆台、镜前、书房、集市、门口审视区是高效穿越场。
- 空间需要兼顾观察、藏拙和立刻试用新认知的可能性。
- 带有旧秩序标识的空间最适合表现主角不适应。

### 对白与节奏
- 主角初期对白宜少说、多试探，避免立刻说出穿帮信息。
- 时代错位句要控制频率，少而准比一直玩梗有效。
- 关键认知差兑现时，先让旁人不懂，再让结果说服他们。

### 分镜表达
- 首次穿越确认要拍清身体、空间和认知三重不协调。
- 重要错位桥段应让主角观察、计算、补救的过程被看见。
- 穿越节奏宜按“异常感知—快速伪装—小试认知差—新的更大风险”推进。

### 生产红线
- 穿越规则、宿主记忆掌握程度与时代知识边界必须前后一致。
- 主角已经学会或暴露过的行为习惯，后续不能随意忘掉。
- 不要把穿越写成换壳重生，忽略世界陌生度。
