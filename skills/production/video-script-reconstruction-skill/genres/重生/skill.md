# 题材补充 Skill：重生

- 重生真正抓人的是“这一次我知道会发生什么，并且我不再那样活”。
- 记忆不是外挂，而是提前布局、提前止损、提前认人的能力。
- 今生的每一次改变，都应该和前世创伤形成对照。
- 卡点优先落在主角第一次改写关键节点、认出旧敌、改变命运方向的时候。

## 人工审核并入经验

### 结构与钩子
- 开局尽快给出前世惨剧或重大错付，让今生目标明确成立。
- 重生后的第一步必须不同于前世，尽早证明主角真的要改命。
- 记忆优势最好落在识破人心、提前止损、改变站队上，而不只是预知天气。

### 人物塑造
- 主角的魅力在于记得痛、敢于改、善于算，而不是永远高冷全知。
- 旧敌旧爱在今生的重新识别，会不断暴露新的情绪层次。
- 重生后的主角应更会隐藏真实意图，而非一直直接挑明。

### 服化道统一
- 今生与前世的阶段感可通过服装状态、发式和色温微调表现。
- 关键道具最好能够跨前世/今生呼应，承担因果锚点。
- 主角造型要保留创伤痕迹或克制感，避免一重生就毫无代价。

### 场景与调度
- 婚宴、议事厅、门口、病榻、签约桌、家宴是高效改命场。
- 对照性空间应固定锚点，方便前世/今生镜头互文。
- 重生戏适合在同一空间里完成“这次不一样”的强反差。

### 对白与节奏
- 重生主角的关键句宜短，重点是让观众听出“我知道你会这样”。
- 旧敌试探和主角反制之间要留足停顿，形成认知压制感。
- 重生题材的真正情绪句应落在前世伤口被重新触发时。

### 分镜表达
- 前世与今生的区别要通过动作选择、站位和表情节奏体现，而不是旁白堆满。
- 关键改命场优先拍主角“提前一步”的判断与行动。
- 重生节奏宜按“回忆锚点—今生改写—对手失措—更大后果”推进。

### 生产红线
- 前世账本必须持续影响今生选择，不能开局提过就失踪。
- 主角已经改写的节点会改变后续因果，不能沿用旧剧情当没变过。
- 不要把重生写成万能预知，削弱布局快感。
