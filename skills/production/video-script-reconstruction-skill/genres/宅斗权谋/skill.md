# 题材补充 Skill：宅斗权谋

- 宅斗权谋看的是礼法表面下的算计、借力、压位和试探。
- 谁坐主位、谁先开口、谁能调动见证人，比情绪宣泄更重要。
- 这个题材靠信息差、名分差、秩序差制造爽感。
- 卡点优先落在局已设好、对手尚未察觉，或真证据即将公开之时。

## 人工审核并入经验

### 结构与钩子
- 开局要尽快让观众看懂当前权力格局、名分关系和潜在对立面。
- 第一轮冲突最好由礼制、称呼、座次或进门资格引爆。
- 每一局必须说清楚谁在试探、谁在设局、谁在借势。

### 人物塑造
- 真正厉害的角色靠节制、观察与借势，而不是一直高声压人。
- 主角要善于利用礼法和长辈意志，而不是只靠个人冲动。
- 反派每次施压都应暴露其急躁、心虚或利益绑定点。

### 服化道统一
- 服饰层级要服务身份、辈分、房头或阵营识别。
- 关键角色的色彩和纹样不宜混淆，避免名分关系读不清。
- 近景妆发要克制，重点保证眼神与微表情的可读性。

### 场景与调度
- 家宴、请安、审问、罚跪、立规矩，是高频有效的权谋场。
- 门槛、台阶和屏风能自然形成进退、主次和偷听结构。
- 主位、旁听位、侍立位要固定，方便连续多集复用。

### 对白与节奏
- 称呼必须准确，辈分与身份错一句都会伤害题材质感。
- 狠话宜借礼法包装，优先“提醒规矩”而不是直接辱骂。
- 关键揭穿句要放在全场静下来之后，增加震慑力。

### 分镜表达
- 先交代全场礼制结构，再落关键人物眼神、茶盏、步伐与站位。
- 发难与接招最好在同一空间连续完成，减少跳场削弱压迫感。
- 冲突推进优先按“试探—压位—逼答—亮证—拍板”组织。

### 生产红线
- 已建立的名分、嫡庶、房头与权力归属不能随剧情随意改写。
- 人证、物证、口供一旦出现，后续必须记得其立场与后果。
- 不要把宅斗权谋写成现代吵架换古装壳。
