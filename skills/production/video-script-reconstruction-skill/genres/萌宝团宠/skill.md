# 题材补充 Skill：萌宝团宠

- 萌宝不是负责卖萌，而是负责撬动成人关系与真相进程。
- 团宠感来自“有人愿意为孩子改变态度和站队”。
- 温情、认亲、护短与误会破解要交织推进，不能只剩可爱桥段。
- 卡点优先落在孩子说破真相、被正式认领、逼出强者出手的时候。

## 人工审核并入经验

### 结构与钩子
- 开局尽快让孩子参与关键关系场，而不是晚半集才出现。
- 孩子的第一轮亮相最好同时完成萌点、判断力和关系撬动。
- 认亲、护短、误会破解、秘密外泄，应围绕孩子交织推进。

### 人物塑造
- 孩子必须推动剧情，而不是只负责说可爱台词。
- 成人角色的魅力很大程度来自他们如何对待孩子。
- 团宠不是所有人都爱孩子，而是关键强者愿意为他承担后果。

### 服化道统一
- 孩子造型优先保证表情与动作灵活，不要被繁复装扮拖累。
- 亲子线中可复用的小物件，如发夹、玩具、挂坠，比大场面更有记忆点。
- 成人的服装区分要清楚，方便孩子与谁一组一眼可读。

### 场景与调度
- 门口拦截、饭桌冲突、医院探视、学校接送、家宴宣布是高效亲子场。
- 空间要允许孩子突然插话、闯入或牵手，直接改写调度。
- 核心空间最好有一两个孩子专属锚点，便于持续建立归属感。

### 对白与节奏
- 孩子的关键台词要短、准、真，避免过度成人化长篇输出。
- 成人对孩子的应答应暴露其真实立场，不能只做功能过场。
- 认亲、护短、道歉类情绪句后要留反应拍，给温情落地。

### 分镜表达
- 孩子的动作、眼神和打断时机要拍清，不能只把他当背景。
- 成人因孩子改变立场时，要给足前后反差。
- 萌宝节奏宜按“可爱动作—有效判断—成人反应—关系改写”推进。

### 生产红线
- 孩子已经触发过的关系变化，后续不能无痕消失。
- 孩子掌握的线索和认知水平要前后一致，不能忽聪忽笨。
- 不要把萌宝写成只会卖萌的吉祥物。
