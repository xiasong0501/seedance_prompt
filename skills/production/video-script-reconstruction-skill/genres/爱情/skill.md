# 题材补充 Skill：爱情

- 爱情题材首先要让观众看见彼此如何吸引、如何误解、如何靠近。
- 真正有效的戏眼来自关系状态变化，而不是外部事件替代情感推进。
- 眼神、沉默、称呼、物件和时机，比宏大宣言更重要。
- 卡点优先落在关系被重新命名、旧伤被碰到、心意暂时无法完成的时候。

## 人工审核并入经验

### 结构与钩子
- 开局尽快建立二人当前关系状态与最核心的情感障碍。
- 第一轮心动最好通过具体事件、物件或称呼错位显形，而不是直接讲爱。
- 每一轮靠近都要对应新的误解、外力或价值冲突，避免平推。
- 爱情戏的情绪支付点常来自一方公开命名关系、另一方默认认领
### 人物塑造
- 两个人都必须是主动者，只是主动方式不同。
- 人物吸引力来自稳定偏好、旧伤痕迹与对彼此的独特反应。
- 感情线成立的前提是两人能真正改变对方的选择。
- 衰败权贵角色应在外形上体现经济衰退（衣料陈旧/颜色暗淡）与精神疲惫（皱纹/眼神浑浊）
- 低身份关键角色外形应先普通可信，再通过眼神/动作建立异质能力
- 隐世强者需‘静中有变’表情管理，避免面瘫；眼神压迫比台词更有力
- 强者人设不靠台词吼叫，而靠‘极简动作+沉默压迫’建立
### 服化道统一
- 造型应服务关系阶段，冷淡、试探、亲密、破裂要能看出差别。
- 情感道具要可复用，如戒指、围巾、旧照片、便签、钥匙。
- 近景妆发必须保证眼神和嘴角细微变化可被读到。
- 女主若要快速推进关系，可借外部威胁触发‘假身份公开认领’装置
- 主角妆容必须极简，确保近景表情可读（尤其眼神变化）
- 衰败角色面部皱纹明显+眼神浑浊, 体现精神负重
### 场景与调度
- 门口、楼梯、车内、餐桌、长椅、病房外走廊都是高效关系空间。
- 空间需要允许两人靠近又被打断，制造未完成感。
- 公开空间承接认领，私密空间承接伤口与试探。
- 开阔户外空间适合做公开羞辱与公开认领
### 对白与节奏
- 爱情对白宜短而含蓄，把真正重要的话留到动作或停顿之后。
- 称呼变化、重复句和答非所问，往往比直白告白更有力。
- 关系转折句后必须留反应拍，不能急着用外部剧情冲掉。
- 男主认领台词要简短有余味，避免长篇表白破坏利落感
- 主角回应前必有0.2秒眼神停顿，体现思考过程
- 关键信息（如‘第九封’）由旁人说出，主角沉默接收，强化反差
### 分镜表达
- 双人关系戏优先保留同框与距离变化，减少只拍单人说话。
- 重要情绪点要给反应停顿，让观众读到未说出口的部分。
- 爱情桥段要多用“靠近—犹豫—打断—回望”而不是直接结果。
- 情绪兑现点要留半拍给男主认领反应，不要台词连珠炮冲掉暧昧
- 女主官宣关系时必须给身体靠近和眼神确认镜头
- 结尾卡点：侧脸特写占2/3画面 + 眼神骤冷 + 字幕淡入 + 停帧2秒
- 集尾按钮必须包含视觉动作（持帖）+表情变化（眼神骤冷）+文字卡点（字幕）
- 结尾卡点：主角侧脸特写+眼神变化+字幕淡入+停帧2秒，不加解释
- 高潮揭示：先全景扫数→特写异常点→反应近景滞后→主角沉默收帖
- 权力反转段：俯拍跪姿→仰拍主角→特写对方惊惶眼神，三镜头闭环
### 生产红线
- 已建立的亲密、裂痕、误会或认领，后续都必须持续产生后果。
- 角色已经暴露过的伤口与偏好，不能后续完全失踪。
- 不要用无穷无尽的误会代替真实关系推进。
- 不要把爱情推进写成突兀真情告白，原集成立基础是策略性公开认领
- 不要用无穷无尽的误会代替真实关系推进；本集‘已故者邀约’是主动牵引，非被动巧合
- 主角沉默但动作具压迫性的特质不可改为频繁台词输出
