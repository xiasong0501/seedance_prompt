# 题材补充 Skill：先婚后爱

- 先把关系绑定逻辑写清楚，再写情感松动与立场变化。
- 这个题材的爽点是偏爱、护短、吃醋与公开认领，不是突然告白。
- 同处一室、共同任务、外部压力，是关系升级最有效的发动机。
- 卡点优先落在“关系被坐实”或“心意即将说破”的前一秒。

## 人工审核并入经验

### 结构与钩子
- 开局必须交代成婚原因、退出代价与双方最初抗拒点。
- 第一轮关系推进优先来自共同处境或共同任务，而不是无来由心动。
- 每集至少安排一次公开护短、站队或偏爱动作，作为情感兑现。

### 人物塑造
- 男女主都要有独立利益与边界，不可只为谈情服务。
- 真正的动心来自立场变化与持续可靠，而非一次性英雄救美。
- 嘴硬、逞强、吃醋、补偿行为应构成稳定的情感阶梯。

### 服化道统一
- 初期色彩与材质要分离，升温后再逐步出现共享色点或柔和质感。
- 婚约、戒指、领带、围巾等关系道具要稳定出现并可回收。
- 妆发要允许近景捕捉克制情绪，不能厚重到吃掉表情。

### 场景与调度
- 餐桌、走廊、车厢、卧房门口、病床边是高效情绪场。
- 同一空间内通过距离、坐立、谁先离开来表现关系变化。
- 公共场合承接护短与认领，私密场合承接松动与坦白。

### 对白与节奏
- 冷话要短，软话要晚，真正心意往往落在动作之后。
- 试探台词要带具体对象与具体事件，避免空泛拌嘴。
- 关键认领句后必须留反应停顿，不能马上转下一轮冲突。

### 分镜表达
- 关系变化优先用距离、站位和谁掌握空间边界来表达。
- 护短时先给截断外界压力的动作，再给被保护者反应。
- 关系推进优先采用“对抗—停顿—靠近—反应”四拍结构。

### 生产红线
- 契约、婚约、隐婚或家族要求一旦设立，不能无代价消失。
- 已经发生的护短与认领会改变双方预期，后续不能装作没发生。
- 不要把先婚后爱写成单纯霸总控制或单方羞辱。
