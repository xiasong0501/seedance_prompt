# 题材补充 Skill：奇幻

- 奇幻的关键是熟悉世界被异常规则轻微撬开，而不是整套修炼体系。
- 惊异感必须可被感知、可被验证、可反过来影响人物选择。
- 奇物、诅咒、预言、命运契约，比大规模世界讲解更有效。
- 卡点优先落在规则刚被证实、后果已出现但解释还不完整的时刻。

## 人工审核并入经验

### 结构与钩子
- 开场尽快让观众看到一个可感知的异常现象，且它发生在日常秩序中。
- 规则揭示要遵循“异常出现—误判—验证—新判断成立”的顺序。
- 每集至少推进一次规则的适用范围、代价或误解后果。
- 奇幻主线推进：拜帖-邀约-赴约三段式结构，每段保留数量/署名/时间异常
- ‘数量异常+署名震惊’结构适用于任何‘规则打破者’登场
- 悬念构建：数量异常 > 内容异常；署名悖论 > 动机模糊
- 权贵围猎模板：质疑者→崩溃者→求饶者情绪三阶递进
- 奇幻宅门题材中，主角的超常能力最好直接解释她为何必须留在这个家，而不是只用来打架
### 人物塑造
- 主角对异常的反应应经历否认、试探、利用或恐惧等真实过程。
- 守秘者不能只是说明书，他必须对规则有自己的立场。
- 被规则影响的人要有清晰情感后果，避免沦为设定工具。
- 权威压制者需通过‘服饰细节+站姿稳定性+目光压迫角’三要素建立识别，而非仅靠台词音量；高阶角色必须有1个可OCR识别的铭文道具
- 边缘助力角色（如二师姐）应外形克制、道具象征、语言理性，避免过早实体化削弱主角孤独感
### 服化道统一
- 造型以现实或熟悉世界为底，异质感通过一两处异常符号建立。
- 奇物或被诅咒对象的材质、磨损、旧化程度要可近看。
- 妆发不宜过于奇装异服化，否则会削弱“异常侵入日常”的对比。
- 私密空间异能揭示模板：生理细节（锁骨纹）+ 日常行为（睡眠习惯）+ 关系试探（发誓）
- 跨集识别锚点三原则：1) 服饰主色不变 2) 关键道具纹样一致 3) 动作逻辑复现
- 高阶角色服饰需有1个可OCR识别的铭文道具（如玉簪刻字、竹简铭文），作为跨集连续性锚点；材质避免过度反光
- 高阶角色服饰需有1个可OCR识别的铭文道具（如玉简、腰牌），作为跨集连续性锚点
### 场景与调度
- 熟悉空间里的一处不对劲，比全场都怪更有效。
- 空间需要留出观察点、验证点和逃离点，服务规则试探。
- 边界、倒影、封闭角落和旧物堆积区，是高频奇幻触发位。
- 奇幻宅门题材中，主角超常能力应直接解释其为何必须留在该空间
- 三人同场必用三角站位：主角居中/跪者低位/质疑者侧立
- 群像戏采用扇形/半跪式分布,确保主角居中高位
- 权威道具（圣旨/玉玺）需特写处理，朱印/金纹清晰可辨；烛光为情绪调节器：摇曳=不安，稳定=伪装平静，骤亮=认知颠覆
- 权威道具（圣旨/玉玺/青瓷碗）需特写处理，朱印/金纹/开片纹清晰可辨，作为跨集识别锚点
- 关键道具最好先在私密空间出现（如本集水桶日常使用），再在公共空间被命名（如‘零米’羞辱）
- 道具真相否定策略：集尾用‘X其实不是Y’句式+视觉证据，替代直接解释
- 命运变量型道具（如聚宝盆）首次揭示必须伴随‘认知差’反转
- 瞳孔反光绑定道具纹路可强化认知反转可信度
- 道具车位于人物后侧或旁侧时，能同时服务身份展示与倒地反差
### 对白与节奏
- 规则说明必须晚于观众可见证据，避免开口就是设定说明书。
- 奇幻对白应优先命名后果，再命名规则本身。
- 真正的解释句要短，留更多空间给误解和反应。
- 奇幻设定揭示避免长篇解释，用短句+视觉配合（如‘淡淡的死期’字幕）
- 奇幻设定解释通过‘旁人质疑→主角无视’对比呈现，避免直接说明
- 群体对话中，质疑句与喝止句间隔≤0.5秒，强化现场感
- 传播型金句段应减少切镜频率，保证口型、字幕和情绪完整落地
- 情绪结算句应放在第一轮惩戒之后，形成从身体打脸到语言打脸的升级
- 手部叙事法：抠抓=压抑，松开=放弃，握拳=决断，伸展=试探，需与台词严格同步
- 用‘道具磨损细节’替代台词说明身份（如水桶结晶=长期挑水）
- 知识性台词（如柴木分类）需在关键词处插入道具特写，避免纯听觉输入
### 分镜表达
- 异常显影优先借环境、道具、影子、声音和反应完成，不急着上大特效。
- 第一次验证规则时，要把主观惊异和客观证据都拍清楚。
- 异常首次显现时，优先展示人物误判和迟疑，而不是马上解释。
- 拜帖/信件类道具特写需包含‘纸张纹理+封印细节+署名字迹’三要素
- 质疑者发言时，镜头保持中景但轻微手持晃动，增强现场不安定感
- 主角高光时刻必须先拍脸后拍动作，突出从容而非蛮力
- 集尾卡点优先用‘半明半暗构图+信息留白字幕’，而非动作高潮，适配奇幻题材认知差
- 关键帧必须服务‘微表情-手部-道具’三角验证，例如‘握拳’需同步展现‘指甲压掌心’‘袖口褶皱’‘烛光投影’
- 转场优先用道具承接（砚台→纸→墨→烛），减少硬切
- 关键反转前必须插入道具特写，帮助观众锁定戏眼
- 真相揭示段禁用特效，优先用道具细节变化（锈迹剥落/铭文显现）替代语言解释
- 真相揭示段优先用‘道具特写+瞳孔反光’组合，避免直接大特效
- 知识传授镜头公式：权威者手部动作特写 → 主角视线跟随 → 关键词停顿+道具特写
### 生产红线
- 已经建立的异常规则必须保持自洽，不能为推进剧情随意改口。
- 奇物、诅咒或预言一旦触发，就要在后续持续留下痕迹。
- 不要把奇幻拍成廉价光效堆砌，而没有可感知规则。
- 避免将奇幻感仅依赖廉价特效, 本集优势在于人物异质造型与压制气场
- 不要把奇幻感只做成廉价特效，当前有效点在‘理性场景中出现非常规判断’
- 不要把奇幻感只做成廉价特效, 当前更有效的是人物异质造型与压制气场
- 不要让堂妹在本集显露阴谋意图（如偷看女主反应），破坏初期信任感；反派不能仅靠吼叫，需有身份道具支撑
- 圣旨/玉佩纹样必须一致；若后续出现新道具，需复刻本集青玉螭龙纹样
- 不可将‘已故者’设定解释为‘梦境’或‘幻觉’，破坏奇幻逻辑；所有异常必须有物理痕迹（如玉简积灰）
- 不可将‘已故者’设定解释为‘梦境’或‘幻觉’，破坏奇幻逻辑
- 禁止将‘地摊奇物’直接定义为‘真宝’，必须保持‘可疑但可能有用’的模糊性
- 青玉螭龙纹样（发簪/玉简）为跨集核心锚点，后续新道具必须复刻纹样结构与金线粗细
- 不要让反派只会吼叫而没有身份道具支撑，跑车和服装是其有效组成部分
