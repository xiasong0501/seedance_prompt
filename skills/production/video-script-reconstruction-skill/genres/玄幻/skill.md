# 题材补充 Skill：玄幻

- 玄幻必须有完整超常体系，人物行动要受境界、血脉、宗门、禁制约束。
- 奇观不是目的，真正抓人的是力量等级、身份真相和代价抬升。
- 每次用力都要让观众更明白世界规则，而不是更糊。
- 卡点优先落在境界突破、封印松动、血脉显形、宗门表态的临界点。

## 人工审核并入经验

### 结构与钩子
- 开局尽快交代当前境界差、宗门/家族秩序和主角面临的硬限制。
- 第一轮冲突最好同时暴露力量差与身份差，让世界规则立住。
- 重要法器、血脉、禁制或誓约必须提早埋下，并反复回收。
- 灵根羞辱→隐性馈赠→认知颠覆”三段式结构适用于玄幻底层逆袭题材
- 玄幻废柴流标准结构：高维回忆→身份揭底→杂役任命→隐忍决心
- 私密奇遇+公共定罪双段式结构适用于玄幻底层崛起线
- 宗门羞辱线结构：日常压迫 → 异常事件 → 污名指控 → 理性干预 → 微光兑现 → 伪善胁迫
- 宗门日常题材中，‘劳动细节’（柴/灰/炉）是解构仙人神话的最佳切口
- 权贵围猎模板：质疑者→崩溃者→求饶者情绪三阶递进
### 人物塑造
- 主角的成长必须建立在认知、选择和代价上，而不是只换皮肤与称号。
- 敌手和同盟都应与世界规则发生具体绑定，不能悬空存在。
- 权威角色的压制力来自资源、规矩和层级，不只是脾气大。
- 玄幻感实现优先级：人物异质造型 > 环境光效 > 特效粒子
### 服化道统一
- 服装与法器要服务境界、宗门、出身和修炼路径的识别。
- 超常人物的异质感应通过材质、纹样、符号和法器细节建立。
- 近景妆发要保证威压、疲态、入魔或觉醒等细微状态可见。
- 玄幻题材中，核心身份标识必须为‘小件金属饰物’（符牌/发簪/耳坠），而非整套华服
- ‘指尖悬停0.3s+微尘粒子’为轻玄幻能力触发标准分镜
### 场景与调度
- 试炼场、祭坛、洞府、宗门议事空间和禁地入口是高效玄幻场。
- 空间中需要有法阵、封印、阶层坐席或资源摆位，体现规则。
- 凡俗空间若承接玄幻冲突，必须明确其如何被超常力量改写。
- 权力反转时必须拍摄身体靠近/转身面向等可视动作，不只拍说话
- 宗门底层空间必须包含‘劳动工具三角区’（水桶-井-晾架），便于调度重复动作戏；山崖类场景必须保留‘雾气遮挡远景’

### 对白与节奏
- 世界观术语要少而准，先让观众看懂后果，再补名词。
- 高位压制句优先点明规则、资格和代价，避免空喊威胁。
- 突破、觉醒、立誓类台词要留出仪式感停顿。
- 传播型金句段减少切镜频率, 保证口型、字幕和情绪完整落地
### 分镜表达
- 先交代力量层级与法则约束，再拍出手结果，避免纯感官混战。
- 重要超常现象要让观众看懂由谁触发、对谁生效、付出了什么。
- 玄幻节奏要让“认知升级”和“力量升级”同步发生。
- 权贵被打脸时：必须给清晰反应近景，形成爽点闭环
- 结尾按钮禁用解释性镜头，必须停在‘后果将至、解释未给’的瞬间
- 开头30秒必须完成‘异常动作（悬浮阵）+离谱判断（这不是阵法传授）+群体反应（无声注视）’三信息
- 情绪兑现点要留半拍给接收方反应（如王爷OS），形成爽点闭环
- 集尾卡点优先停在‘后果将至、解释未给’的视觉留白瞬间，而非语言总结
- 反派每次被打脸后都要有清晰反应近景，形成爽点闭环
### 生产红线
- 已建立的境界、法器规则、宗门秩序和血脉限制必须前后一致。
- 任何重要力量跃迁都要有前置条件与后续代价。
- 不要把玄幻写成普通短剧冲突外加一层光效。
- 禁止在玄幻宗门场景中使用现代材质（如不锈钢、塑料）；避免让主角过早展现‘眼神坚定’
- 禁止在玄幻宗门场景中使用现代材质（如不锈钢、塑料）
- 不要把玄幻写成普通短剧冲突外加一层光效；每次出手都应带来可见代价或更高层追责
- 不要把奇幻感只做成廉价特效，当前更有效的是人物异质造型与压制气场
