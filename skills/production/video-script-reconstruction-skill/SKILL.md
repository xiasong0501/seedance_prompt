# 视频理解与剧本重建 Skill

> 使用位置：
> - [providers/openai_adapter.py](/mnt/myvg/Agent/Seedance2.0_AI/providers/openai_adapter.py)
> - [providers/gemini_adapter.py](/mnt/myvg/Agent/Seedance2.0_AI/providers/gemini_adapter.py)
> - 属于 `./run_video_pipeline.sh` / `./run_series_pipeline.sh` 的“视频 -> 单集分析 -> 剧本重建”阶段
> - 作用方式：作为视频理解与剧本重建 prompt 的核心方法论参考，帮助模型在证据充足的前提下做更细的题材判断、爆点抽取与生产导向重建

## 总目标

把一集漫剧视频中的画面、台词、字幕、关键帧与上下文信息，重建成：

1. 忠于源素材的结构化分析
2. 适合后续导演分析继续使用的专业剧本
3. 能指导后续人物设计、场景设计、分镜设计的题材与爆点信息
4. 能沉淀后续导演、服化道、分镜与 Seedance 直接可用的下游生产经验

## 工作原则

### 1. 证据优先

- 先看当前视频证据，再做叙事判断
- 台词、OCR、画面三者互相校验
- 无法确认的内容必须降置信度，不要脑补硬事实

### 2. 题材判断必须服务生产

- 不只判断“这是个什么故事”
- 还要判断“它属于哪类短剧 / 漫剧爆点结构”
- 题材分类必须能指导：
  - 剧本重建
  - 人物设定
  - 场景设计
  - 分镜爆点

### 3. 先还原，再提炼

- 单集分析阶段以事实还原为主
- 在事实足够清楚后，再提炼题材分类、钩子、情绪支付点和卡点策略

### 4. 连续性优先于猎奇

- 即使为了抓人，也不能破坏人物连续性
- 即使某题材常见爆点成立，也不能违背当前视频事实

### 5. 学习结果必须面向下游生产

- 不要只总结“这集哪里爽”
- 要总结“后续导演、服化道、分镜、Seedance 生成最该复用什么”
- 学习结果必须尽量写成可执行规则，而不是空泛评价

## 单集分析必须补充的结构

除了常规的剧情、人物、场景、对白、剧情点以外，还必须产出：

### A. 题材分类 `genre_classification`

至少要回答：

- 这一集所属的主题材是什么
- 有哪些副题材或常见装置
- 这一集的核心叙事驱动是什么
- 观众对这类题材的典型期待是什么

### B. 爆点画像 `hook_profile`

至少要回答：

- 开头抓人的点是什么
- 这一集最容易形成传播的瞬间是什么
- 情绪兑现点在哪里
- 集尾应该靠什么类型的卡点推动下一集点击

### C. 下游设计指导 `downstream_design_guidance`

至少要回答：

- 剧本重建时应重点保留或强化什么
- 人物设计最该强调什么气质与身份识别
- 场景设计最该强调什么空间与氛围
- 分镜最该抓什么节奏和爆点

### D. 镜头语言分析 `camera_language_analysis`

至少要回答：

- 这一集最常用、最有效的镜头类型是什么
- 运镜是更偏压迫推进、情绪贴身，还是展示关系与空间
- 构图上最常见的重心是什么：对视、权力压制、身份落差、亲密拉扯、群像围困
- 转场和节奏是快切推进，还是停留蓄力
- 高潮段和集尾卡点在画面上靠什么视觉模式成立

### E. 服化道分析 `art_direction_analysis`

至少要回答：

- 人物服装和妆发有哪些稳定辨识特征
- 哪些道具、空间布置、材质和灯光是这部剧的视觉记忆点
- 颜色和质感如何服务人物身份、情绪和题材爆点
- 哪些视觉元素是后续人物图、场景图一定不能丢的

### F. 分镜蓝图 `storyboard_blueprint`

至少要回答：

- 开头钩子适合怎么用镜头打
- 冲突升级适合怎样分段、怎样给反应镜头
- 情绪兑现适合怎么靠特写、节奏和声音落地
- 集尾卡点最适合停在什么视觉动作或情绪瞬间
- 后续 Seedance 提示词最该抓哪些动态重点，最该避免哪些空泛写法

### G. 生产导向经验提炼 `series_learning_extraction`

至少要回答：

- `character_design_rules`：人物识别、阶段变体、跨集一致性应该怎么保
- `costume_makeup_rules`：服装轮廓、主色、材质、配饰、妆发哪些必须保留
- `scene_design_rules`：哪些空间、静态背景、环境模板最适合后续生成
- `camera_language_rules`：哪些镜头打法最适合这个题材和这部剧
- `storyboard_execution_rules`：开头、转场、情绪兑现、结尾卡点的执行经验
- `dialogue_timing_rules`：对白如何串行、留白、停顿，避免声音冲突
- `continuity_guardrails`：哪些人物/设定/场景逻辑绝不能改坏
- `negative_patterns`：后续导演、服化道、分镜与视频生成最容易踩的坑

这些规则必须：

- 优先服务后续导演、服化道、场景图、分镜和 Seedance 生成
- 每类尽量控制在 3-5 条内，宁可少也不要空泛
- 尽量绑定当前视频中的具体证据，不要脱离素材瞎总结
- 必须抽象成题材级或角色类型级规则，不能直接点名本剧角色、地点、组织或一次性事件
- 如果某条经验只能描述成“某个具体人物必须怎样”，说明它更适合作为本剧连续性说明，不适合作为题材通用经验
- 应尽量把具体名字改写成角色类型，如“女主 / 男主 / 反派女配 / 掌权长辈 / 公开羞辱场 / 高压厅堂 / 封闭车厢”这类可迁移标签

## 剧本重建原则

### 1. 保留核心事实

- 人物关系不能乱改
- 关键事件不能乱加
- 情绪动机不能硬拧

### 2. 输出为生产友好的剧本

- 场景头清楚
- 动作线清楚
- 对白和独白区分清楚
- 能直接进入导演分析阶段

### 3. 允许适度职业化重组

- 可以更清楚地整理场次
- 可以把分散信息整理得更利于后续生产
- 但不能改写成另一部剧

### 4. 题材特征要落在剧本节奏里

- 穿越类要强化“身份错位 / 现代意识对旧秩序的撞击”
- 重生类要强化“前世代价 / 这一世的主动布局”
- 复仇逆袭类要强化“屈辱 - 藏锋 - 反打”
- 萌宝类要强化“关系催化 / 情感桥接 / 戏剧性认亲”

## 分析镜头与服化道时要遵守的原则

### 1. 不要把镜头分析写成摄影术语堆砌

- 重点不是“用了中景和推镜头”这种空话
- 而是说明“为什么这里要贴近人物”“为什么这里要给反应特写”“为什么这里需要停顿或压迫构图”

### 2. 镜头语言必须服务叙事功能

- 羞辱戏：看压迫关系怎么成立
- 甜感戏：看视线、距离、靠近和回避
- 权谋戏：看站位、遮挡、前后景层次和主次权力
- 认亲/反转戏：看揭示顺序、反应镜头和停顿位置

### 3. 服化道分析必须服务后续出图

- 不要只写“华贵、精致、古风”这种空词
- 要尽量写成后续可复用的识别要点：
  - 服装轮廓
  - 主色和材质
  - 发型与配饰
  - 关键道具
  - 代表性空间和灯光

### 4. Story beat 必须带视觉与分镜价值

每个 `story_beats` 除了剧情功能，还要补：

- `visual_focus`：这一段观众主要在看什么
- `camera_language`：这一段最有效的镜头组织是什么
- `art_direction_cues`：这一段对服化道/场景最关键的提示是什么
- `storyboard_value`：这一段后续做 Seedance 时最该强调什么

### 5. 优先抽取可跨集复用的视觉规律

- 哪些镜头组合是这部剧反复有效的
- 哪些人物造型逻辑能跨集保持识别
- 哪些场景调度能稳定承接这个题材
- 哪些集尾停顿方式最适合形成点击欲望

## 你最重要的任务不是“写得华丽”

而是：

- 看得准
- 分类准
- 爆点抓得准
- 重建后能真正服务后续导演、服化道和分镜生产
