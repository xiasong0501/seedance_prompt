# 人物设定提示词示例

以下示例展示高质量的人物设定提示词写法。每个提示词可直接复制到文生图工具生成角色设定图。

---

## 示例1：都市青年女性（写实风格）

**出图要求**：一张图，左半边面部特写，右半边全身正面、侧面、背面设定图，白色背景

**提示词**：

角色设定图，白色干净背景。画面左半部分是面部特写，右半部分是全身正面、侧面和背面三个角度的设定图，水平排列。

这是一位二十六七岁的中国女性，鹅蛋脸，五官精致但不过分锐利，弯眉杏眼，瞳色深棕近黑，鼻梁挺直但线条柔和，薄唇，肤色白皙偏暖。头发是自然黑色中长发，长度刚过肩膀，微微内扣的发尾，额前有一缕斜分的碎刘海，发质顺滑有光泽。

身高约一米六五，身材匀称偏纤细。上身穿一件米白色圆领羊绒毛衣，质地柔软贴身，袖口微微挽起露出手腕。下身是深灰色高腰直筒西裤，裤腿垂感好，裤脚刚好落在脚踝上方。脚穿一双黑色尖头低跟皮鞋，鞋面干净，跟高约三厘米。

左手腕戴一块银色表盘的极简手表，右手无名指上有一枚细细的银戒指。耳朵上是一对小巧的珍珠耳钉。没有项链，没有其他多余配饰。

整体气质是干练但温和的都市职业女性，站姿端正但不僵硬，眼神沉稳，嘴角微微上扬，有一种自信而亲切的气场。

## 示例3：古风少女（国漫风格）

**出图要求**：一张图，左半边面部特写，右半边全身正面、侧面、背面设定图，白色背景

**提示词**：

国漫风格角色设定图，白色干净背景。画面左半部分是面部特写，右半部分是全身正面、侧面和背面三个角度的设定图，水平排列。

这是一位十七八岁的少女，瓜子脸，面容清秀灵动。柳叶眉细长上扬，一双桃花眼水润明亮，瞳色是偏琥珀的浅棕，带着少女特有的好奇与灵气。琼鼻小巧，唇色自然红润如樱花瓣，下巴尖而圆润。肤色如凝脂，吹弹可破。

发色是墨黑色，长发及腰。头顶用一根翠绿色的玉簪挽起一个松散的飞仙髻，剩余的头发披散在身后，发尾微微卷曲。额前散落几缕碎发，耳侧各有一绺鬓发垂至下巴。发间点缀着两三朵小小的白色绢花。

身材娇小玲珑，约一米六。穿一套浅粉色交领襦裙，上襦是淡粉色丝绸面料，领口和袖口绣着浅金色的梅花暗纹，袖子是宽大的琵琶袖，袖长过手。下裙是渐变色——从腰间的浅粉过渡到裙摆的淡紫，裙摆拖地，裙身绣着几枝写意的白梅。腰间系一条翠绿色丝带，与头上玉簪呼应，丝带末端垂下两根长长的飘带。脚穿淡粉色绣花布鞋，鞋头微翘，鞋面绣着一朵小梅花。

右手腕戴一只温润的白玉镯，左手握着一把折起的淡绿色团扇，扇面隐约可见水墨山水画。

整体气质是灵动而不失端庄的古代少女，站姿轻盈，身体微微侧倾，像是随时会被风吹走，眼中带着对世界的好奇与天真。
