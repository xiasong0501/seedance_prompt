# 场景环境提示词示例

以下示例展示高质量的场景宫格提示词写法。所有场景排列在一张宫格图中，每格一个场景，风格统一。提示词可直接复制到文生图工具生成场景参考图。

## 示例1：都市情感短剧（7个场景，3×3 九宫格，后2格留空）

请生成一张 3×3 九宫格布局的电影场景环境图像。每个格子代表一个独立场景，所有格子必须保持视觉风格统一。请按照以下规范生成。

### 视觉规范
整体风格：真人写实，9:16 竖屏电影感构图，移动端安全区意识强
色彩基调：冷暖对比为主——工作场景偏冷蓝灰，私人空间偏暖黄橙
材质质感：现代都市的玻璃、金属、混凝土质感，夜景有湿润的路面反光

### 宫格布局
第1行：格1-3 / 第2行：格4-6 / 第3行：格7（格8-9留空）

### Panel Breakdown（场景拆解）

格1——【深夜写字楼办公室】
视角：现代都市写字楼的开放式办公区，深夜十一点。大部分工位已经空了，椅子被推开，桌面上散落着文件和未合上的笔记本电脑，屏保的微光在黑暗中闪烁。只有最靠窗的一排还亮着台灯，暖黄色的光在桌面上画出一个个孤独的圆。落地玻璃窗外是城市夜景，远处高楼灯光如星，窗户上映着室内倒影。天花板荧光灯已关，只有走廊尽头安全出口标志发出暗绿色的光。整体色调是冷蓝灰与暖黄的对比。
重点：台灯暖光与窗外冷光的反差，加班后空旷的疲惫感

格2——【老旧居民楼楼道】
视角：九十年代南方城市的老式六层居民楼楼道内部，傍晚时分。水泥墙面泛黄，贴着褪色的社区通知和小广告。楼梯扶手是铁质的，漆面斑驳露出底下的锈色。每层楼梯转角处有一盏声控灯，发出惨白的荧光，照亮窄小的空间。各家门口堆着杂物——折叠自行车、纸箱、旧鞋架。某一户的门半开着，暖黄色的灯光和炒菜的油烟从门缝里漏出来。
重点：老旧楼道的生活气息，声控灯的惨白光与住户透出的暖光对比

格3——【街边便利店】
视角：城市街角的24小时便利店，深夜两点。店面玻璃门上贴着促销海报，店内日光灯管把整个空间照得通亮刺眼，与外面漆黑的街道形成强烈反差。货架排列整齐，零食、泡面、饮料分区陈列。收银台旁的关东煮锅冒着热气，玻璃罩里的包子已经干瘪。靠窗的高脚吧台区有两把红色塑料高凳，台面上放着吃了一半的便当和一罐啤酒。门外的人行道上，地面被店内灯光染成一片暖白色。
重点：深夜便利店的日光灯通亮感，橱窗内外的明暗对比

格4——【出租屋卧室】
视角：城中村的小单间出租屋，夜晚。不到十五平米的空间塞满了生活的全部——单人床靠墙放着，床上堆着没叠的被子和几件衣服。床头是一张折叠小桌当书桌用，上面摞着书和一台旧笔记本电脑，屏幕的蓝光是房间里唯一的光源。墙上贴着几张打印的照片和一张城市地图。窗户很小，挂着薄窗帘，窗外是对面楼的外墙，几乎没有视野。角落里有一个简易衣架，挂着几件衬衫和一件外套。
重点：逼仄空间里的电脑蓝光，小房间塞满生活痕迹的局促感

格5——【天台】
视角：老式居民楼的天台，凌晨。水泥地面上散落着几个废弃的花盆和一根生锈的晾衣杆。天台边缘是一米高的水泥矮墙，墙面有水渍和青苔。站在天台可以俯瞰整个城中村的屋顶——密密麻麻的铁皮顶和水塔，远处是城市CBD的高楼群，灯火辉煌。天空是深蓝偏紫的，最亮的几颗星勉强可见。凌晨的风把矮墙上晾着的一条毛巾吹得微微飘动。
重点：天台的开阔感与城中村/CBD的远近对比，凌晨天空的深蓝色调

格6——【咖啡馆】
视角：文艺风格的独立咖啡馆，下午三点。阳光从落地窗斜射进来，在木质地板上投下窗框的影子。吧台是深色木质的，上面摆着一台铜制手冲壶和几个陶瓷杯。墙面是暖灰色的水泥质感，挂着几幅黑白摄影作品。角落的书架上摆满了书，旁边是一把旧皮沙发。店里放着爵士乐，吧台后面的黑板上用粉笔写着今日特调。靠窗的位置有一张双人小桌，桌上放着一杯拿铁，咖啡表面的拉花还没散。
重点：午后斜射阳光的温暖感，木质与水泥的材质对比

格7——【雨夜街头】
视角：城市主干道的人行道，秋天的雨夜。路面被雨水打湿，积水中映着红绿灯、车灯和商铺霓虹的倒影，色彩斑斓地铺在黑色柏油路上。人行道旁的梧桐树落了一地黄叶，湿漉漉地贴在地砖上。路边停着一排共享单车，车座上积着雨水。远处的十字路口，红灯在雨幕中晕开一团红色光晕。一个公交站台的广告灯箱还亮着，白色的光照亮了站台下一小片避雨的空间。
重点：雨水路面的霓虹倒影，红绿灯在雨中的光晕

（格8-9留空）

## 示例2：古装悬疑（5个场景，3×3 九宫格，后4格留空）

请生成一张 3×3 九宫格布局的电影场景环境图像。每个格子代表一个独立场景，所有格子必须保持视觉风格统一。请按照以下规范生成。

### 视觉规范
整体风格：国漫水墨与写实结合，电影感构图，色彩饱和度偏低
色彩基调：以墨色、灰白、青绿为主，点缀暗红作为视觉锚点
材质质感：水墨渲染的山石树木，木质建筑有年代感的肌理

### 宫格布局
第1行：格1-3 / 第2行：格4-5（格6-9留空）

### Panel Breakdown（场景拆解）

格1——【官府大堂】
视角：唐代县衙的审判大堂，白天。高大的木质立柱撑起沉重的梁架结构，柱身刷着暗红色的漆，漆面已有细密的龟裂。堂上高悬"明镜高悬"牌匾，金字在阴影中若隐若现。公案桌上摆着惊堂木、笔架和一摞卷宗。堂下是青石板地面，两侧站班衙役的位置放着水火棍。光线从大堂正门射入，在地面上投下方形的光区，越往里越暗，形成明显的明暗纵深。
重点：大堂的纵深明暗对比，暗红立柱与青石板的庄重感

格2——【密室书房】
视角：隐藏在宅邸深处的密室书房，烛光。狭小的空间四面都是木质书架，塞满了竹简和线装书册。中央一张矮案，案上摊开一幅地图，旁边是一盏青铜油灯，灯焰微微摇晃，在墙面和书架上投下摇曳的影子。案角压着一枚玉印和几封密信。地面铺着草席，角落里有一个铜炉，炉中的熏香已经燃尽，只剩一缕青烟。整个空间被烛光染成暖黄色调，但四周的暗角透着不安。
重点：烛光在密室中的摇曳光影，狭小空间的紧迫感

格3——【山间驿道】
视角：秦岭山间的石板驿道，黄昏。窄窄的石板路沿着山腰蜿蜒，一侧是长满青苔的岩壁，另一侧是深不见底的山谷，谷中白雾翻涌。路旁有一座破旧的石亭，亭柱上刻着模糊的文字，亭内的石凳上有旅人留下的干粮碎屑。远处山峦层叠，夕阳从云层的缝隙中射出几道金色光柱，照亮了山坡上一片金黄的银杏林。石板路面湿滑，有马蹄踏过的痕迹。
重点：驿道的纵深蜿蜒感，黄昏光柱穿过云层的戏剧性光影

格4——【酒肆】
视角：小镇上的二层木质酒肆，正午。一楼大堂摆着七八张方桌，桌面磨得发亮，上面有酒渍和刀痕。柜台后面是一排酒坛，坛口用红布封着，墙上挂着酒旗和价目木牌。楼梯是木质的，踏板中间凹陷，被无数脚步磨出了弧度。正午的阳光从敞开的大门和二楼窗户照进来，光线中浮着灰尘。一张靠角落的桌上放着吃了一半的菜和一壶酒，好像客人匆匆离去。
重点：酒肆的市井烟火气，正午阳光中浮尘的质感

格5——【地牢】
视角：县衙地下的牢房，几乎无光。粗糙的石壁上渗着水珠，地面是潮湿的泥土，角落里铺着发霉的稻草。铁栅栏门锈迹斑斑，锁链垂在两侧。唯一的光源是走廊尽头一盏快要熄灭的油灯，微弱的光勉强照出牢房的轮廓，铁栅栏在地面投下平行的条状阴影。牢房深处完全是黑暗，只能隐约看到墙上有被指甲划过的痕迹。空气中弥漫着潮湿、霉变和铁锈的气息。
重点：极暗环境中微弱油灯的光影，铁栅栏的条状阴影

（格6-9留空）
