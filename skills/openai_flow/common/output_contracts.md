# 输出契约 Skill

> 使用位置：
> - 由 [flow.py](/mnt/myvg/Agent/Seedance2.0_AI/openai_agents/flow.py) 作为 `common_docs` 注入给 `Producer / Explosive / Director / Art / Storyboard` 全部 Agent
> - 作用方式：定义每类核心产物最少必须包含什么，防止 agent 只写概念化内容

## 导演分析契约

必须包含：

- 剧情结构总览
- 按剧情点拆解的导演讲戏
- 人物表
- 场景表
- 素材状态：`新增 / 复用 / 变体`

## 服化道契约

人物提示词必须：

- 能支撑参考图生成
- 清楚定义脸、身形、发型、服装、配饰、鞋履、气质
- 明确设定图式输出版式

场景提示词必须：

- 能支撑多格宫格图生成
- 清楚定义空间、光照、色板、材质、道具、氛围
- 保持各格视觉统一

## 分镜契约

必须：

- 一条剧情点映射一条提示词块
- 以导演意图为第一真实来源
- 需要时正确整合素材引用
- 保持对 Seedance 风格视频生成可执行

## 爆款改稿契约

必须：

- 给当前版本打分
- 说明优点与短板
- 说明具体改稿计划
- 明确列出“爆款升级加在了哪里”
- 保持主线剧情连续性
