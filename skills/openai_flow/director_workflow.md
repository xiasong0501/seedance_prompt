# 导演 Agent 工作流 Skill

> 使用位置：
> - 由 [flow.py](/mnt/myvg/Agent/Seedance2.0_AI/openai_agents/flow.py) 注入给 `Director Agent`
> - 属于 `./run_openai_agent_flow.sh` 的导演阶段编排说明
> - 作用方式：定义 Director Agent 应如何把剧本转成 `01-director-analysis.md`

## 目标

把剧本转成尽可能强的 `01-director-analysis.md`。

## 必需流程

1. 仔细阅读剧本
2. 识别自然剧情点
3. 为每个剧情点给出合理时长建议
4. 为每个剧情点写导演视角的讲戏
5. 提取可直接用于生产的人物表和场景表
6. 为每个人物和场景标记 `新增 / 复用 / 变体`

## 剧情点质量要求

每个剧情点都应包含：

- 谁在场
- 在哪里
- 发生了什么动作
- 发生了什么情绪变化
- 镜头感觉
- 光影 / 色彩 / 氛围

## 导演分析标准

- 剧情点必须可拍、可生成、可执行
- 切分边界应服从叙事目的，不服从任意换行
- 这份文档必须能直接帮助服化道和分镜阶段继续工作
