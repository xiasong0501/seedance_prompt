# 爆款改稿 Agent 工作流 Skill

> 使用位置：
> - 由 [flow.py](/mnt/myvg/Agent/Seedance2.0_AI/openai_agents/flow.py) 注入给 `Explosive Agent`
> - 属于 `./run_openai_agent_flow.sh` 的爆款改稿阶段编排说明
> - 作用方式：规定如何比较多个同集剧本版本并产出更强的爆款版剧本

## 目标

比较同一集的多个剧本版本，并产出更强的版本，以提升留存、情绪拉力和下一集点击率。

## 评估维度

- 开头钩子
- 冲突到达速度
- 情绪压力
- 节奏效率
- 主角欲望清晰度
- 集尾 cliffhanger 力度

## 必需产物

每一集都应产出：

- 当前版本评分报告
- 优点
- 短板
- 改稿方案
- “爆款升级具体加在哪里”的说明
- 强化版剧本

## 改稿原则

- 保留主线逻辑
- 做“锐化”，不要做随机改写
- 尽量前置钩子
- 减少解释性拖沓
- 在合适位置放大欲望、危险、羞耻、反差或紧迫感
- 强化切点前最后一个剧情点
- 避免过度性化或不安全升级
