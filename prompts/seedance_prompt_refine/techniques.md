# Seedance Prompt Refine 技巧

- 只优化提示词，不改故事事实、人物关系、事件顺序和剧情点数量。
- 一条提示词只承载一个连续叙事段落，不把无关内容硬塞进同一条。
- reference 已经提供的人物长相、服装、场景静态布局不要重复堆写，重点写变化、动作、调度、情绪和镜头运动。
- 优先检查 @图片 引用是否错指、漏指、混指；角色和场景引用要和当前条的主动作、主空间一致。
- 每条都要交代当前在镜角色、谁从哪里入画、谁留在前景/中景/后景、谁如何退场或被遮挡。
- 角色要先入画、转头、抬手、靠近或被镜头带到，再说话、出手或占据特写，避免凭空出现。
- 人物与背景的空间关系要稳定：前中后景、高低位、远近、屏幕左右、视线方向、手位与道具位置尽量清楚。
- 相邻分镜要有动作交棒、视线交棒、声音交棒、特效余波交棒或尾帧姿态交棒，减少断裂。
- 场景切换要有动机：由动作推进、空间转移、视线切换、声响触发或结果落点触发，不要硬跳。
- 同一地点的不同区域切换，要保留统一空间母体与轴线感，不要像换了一个新场景。
- 相邻分镜不要反复重复同样的建立镜头、同样的情绪判断、同样的动作说明和同样的气氛词。
- 冗余信息优先压缩，把字数让给真正推进剧情、强化画面和增强执行性的内容。
- 画面节奏要紧凑但不能跳步，每 2-3 秒最好有新动作、新信息、新情绪或新结果。
- 高价值节点优先形成“规模镜头 -> 人物近景/特写 -> 结果/后果落点”的重音梯度。
- 运镜要绑定稳定空间锚点和明确路径，不要只写“推近、拉远、切一下”这类空话。
- 特效要写成物理事件：源头、传播路径、受光变化、烟尘/碎屑/余波、人物与环境反馈都尽量可见。
- 关键道具在相邻分镜之间要连续：位置、朝向、手位、状态、受光和损耗不要跳变。
- 细节要有吸引力但不能浮夸：优先补足表情变化、动作层次、材质反馈、光影落点、声音钩子和情绪兑现。
- 提升质感时，优先写可执行的镜头语言，不要写空泛文学感套话。
