终审优先级：
1. 导演讲戏本与 `story_point` 覆盖完整性。
2. 人物与引用绑定准确性。
3. 分镜内部的动作、空间、物理与镜头逻辑。
4. 跨分镜的连续性、镜头衔接和视觉一致性。
5. 细节密度、可视化因果和平台稳定性。

必须完成的检查与修订：
1. `prompt_entries` 的数量、顺序、`point_id`、`title` 必须仍与导演 `story_point` 一一对应；不得漏掉后半段，也不得私自合并。
2. 若导演原文中的关键对白、反击、揭示、关系变化、情绪转折、危险靠近或反应被吞掉，必须补回来。
3. `primary_refs` / `secondary_refs` 中保留的每个 @图片编号，都必须在该条 `prompt_text` 中原样出现；若没真正使用，要么补正文绑定，要么删 ref。
4. 如果正文用了某个 @图片编号，但 refs 里没有，必须补齐。
5. 若正文中核心人物已经承担主要动作或对白，但 ref 绑定模糊、错指、漏指或混指，必须修正。
6. 若场景类 ref 被保留，`prompt_text` 必须真正落下具体环境信息，而不是只写泛泛的地点标签。
7. 若一整集本来有可用场景参考，最终却几乎没被实用化，必须补足，而不是放任通过。
8. `dialogue_timeline` 必须串行；若存在双声叠台词、同秒冲突或对白顺序与画面顺序打架，必须重排。
9. `continuity_bridge` 必须精简到 1-2 句，但必须真的给出跨条连续锚点，不能只写“冲突升级”“情绪延续”。
10. `prompt_text` 必须是成片正文，不能出现字段复述、workflow 口吻、模板化尾注或解释性废话。
11. 若 `prompt_text` 与 `dialogue_timeline`、`shot_beat_plan`、`audio_design` 冲突，优先修正 `prompt_text`，确保最终喂给 Seedance 的文本是自洽的。
12. 若某条只写了结果、没写前提或中间桥，导致观众难以理解因果，必须补足最短但关键的可视化桥接。
13. 若某条存在手部占用冲突、姿态前提缺失、同一道具状态跳变、空间位置跳变、人物突然出现或突然消失，必须修正。
14. 若出现大范围攻击、爆炸、雷击、群体冲击、强压制或大场面结果后，关键人物或关键道具仍持续存在，但没有解释其安全区、遮挡、留空、避让、护持或未被正面波及的依据，必须修正。
15. 若高能效果只剩“炸开、爆闪、齐鸣、雷落下、冲击掀开”等抽象词，而没有源头、传播路径和环境/人物反馈，必须补成三层可执行描述。
16. 若某条的 `shot_beat_plan` 只有剧情摘要，没有明确的人物-镜头-空间关系，必须重写得更具体。
17. 若某条同一时间调度过多主角，导致前景主体混乱，必须收束到 1-4 个核心主体，其余只保留背景反应、遮挡层次或秩序信息。
18. 若某条只有中段动作，没有首帧承接状态或尾帧落点，必须补齐，让 Seedance 能自然接前后条。
19. 必须逐条检查镜头是否有明确动机和路径：推近、回拉、切视角、顺势跟拍都要由动作、视线、声响、道具或特效触发。
20. 必须逐条检查 beat 与 beat 之间是否有因果链；若只是并列罗列，必须改成连续推进。

跨分镜镜头逻辑必须重点终审：
1. 上一条尾帧、当前条首帧、下一条首帧之间的主体朝向、视线关系、主光方向、屏幕方位、站位轴线、关键道具位置、空间锚点和情绪余波应尽量连续。
2. 视角切换必须由前一拍的动作、视线、声响、空间压力或冲击触发，不能无动机瞬移或跳轴。
3. 同一场戏内不能频繁更换镜头高度、受光逻辑、空间比例、人物体量和镜头语言，避免像不同项目组分别生成。
4. 若某条结尾无法自然把视线、动作、道具或空间落点交给下一条，必须修出明确的交棒状态。
5. 若下一条首帧和上一条尾帧之间缺少可见桥接，优先补最短但最关键的连续锚点，而不是大段重写剧情。

细节密度与描述质量也必须终审：
1. 若某条只剩抽象情绪和剧情摘要，应补 1-3 个有价值的物理细节，例如受风、反光、石屑、火星、呼吸、脚步、器物轻震、烟尘层次、受光变化、边缘光或材质反馈。
2. 细节必须服务可视化，不要为了“变厚”而堆砌修辞。
3. 若核心人物处在高压、濒死、翻盘、施术、压制、交接关键道具或决定性对视状态，后续 beat 必须持续确认其身体状态、视线意图、道具手位或站位关系，防止人物被场面吞掉。
4. 若某条看起来“词很多但信息很少”，必须重写到信息增量更高，而不是单纯增字。

节奏、构图与生成稳定性：
1. 必须检查 `pace_label` 与 `density_strategy` 是否合理：不能为了快而牺牲关键叙事，也不能为了稳而把段落拍空。
2. 必须检查是否真正按目标画幅构图；若目标是 9:16 竖屏，主体、手部动作、道具和爆点不能长期落在左右边缘。
3. 群像、对峙、入场和高能段，必须优先组织成适合竖屏的前中后景分层、高低位和纵深关系。
4. 若某条必须靠超宽横移才能看清关系，说明构图还不够适合目标画幅，必须修正。

风格一致性与平台收束：
1. 必须逐条检查跨分镜视觉一致性：同一角色与同一场景反复出现时，主光方向、色温、对比度、人物体量、镜头高度、服装层次、材质质感和空气透视不能轻易漂移。
2. 必须重点排查可能触发平台高风险的画面表达；如存在，优先保留剧情结果、关系变化和镜头节奏，删掉最敏感的细节化描写。
3. 不要无故扩写，不要把 prompt 改得更空，也不要换成另一套文风。
